# Changelog

---

## 0.3

- Added support for scanning token store keys on Vista using [vistaspctl](https://github.com/InvoxiPlayGames/vistaspctl) method by @InvoxiPlayGames
- Added support for .xrm pkeyconfig file types (found in MapPoint).
- Added random digits to exported CSV filenames because Excel can't open multiple files with the same name.
- Fixed some bugs.
- Added pkeyconfigs for the following products:
  - Windows Embedded Standard 7 Toolkit
  - Windows Embedded Standard 8 Toolkit
  - Forefront Protection 2010 for Exchange Server
  - Forefront Protection 2010 for SharePoint
  - Forefront Threat Management Gateway 2010
  - Flight Simulator X
  - AutoRoute 2010
  - AutoRoute 2011
  - AutoRoute 2013
  - MapPoint 2009
  - MapPoint 2010
  - MapPoint 2011
  - MapPoint 2013
  - Streets and Trips 2008
  - Streets and Trips 2009
  - Streets and Trips 2010
  - Streets and Trips 2011
  - Streets and Trips 2013

**Note**: Retail pkeyconfig files for some versions of AutoRoute, MapPoint, and Streets & Trips are still missing. However, volume pkeyconfig files have been added for all of them.
