## Installation Notes

- Windows PowerShell 5.1 is the last version in the 5.x line. Modern cross-platform PowerShell (PowerShell 6 and later) is a separate product and is not covered here.
- Each WMF package includes the corresponding PowerShell version. For example, WMF 5.1 installs PowerShell 5.1.
- WMF 3.0 must be uninstalled before installing WMF 5.1 on Windows 7 and Server 2008 R2, as having both installed leads to [bugs](https://learn.microsoft.com/en-us/previous-versions/powershell/scripting/windows-powershell/wmf/setup/install-configure?view=powershell-7.3#wmf-51-prerequisites-for-windows-server-2008-r2-sp1-and-windows-7-sp1).
- WMF 4.0 must be installed before WMF 5.0 on Windows 7 and Server 2008 R2, as skipping it leads to [bugs](https://web.archive.org/web/20171208233034mp_/https://docs.microsoft.com/en-us/powershell/wmf/5.0/requirements).

| Windows Version                                   | Preinstalled PowerShell Version | Latest Supported PowerShell Version                                                                                                                                                                                                                                      |
| ------------------------------------------------- | ------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| Windows 10 1607 and all newer versions            | **5.1**                         | NA                                                                                                                                                                                                                                                                       |
| Windows 10 1507 / 1511                            | **5.0**                         | None                                                                                                                                                                                                                                                                     |
| Windows 8.1                                       | **4.0**                         | **5.0** <br /><br /> **5.1** (requires .NET Framework 4.5.2)                                                                                                                                                                                                             |
| Windows 8                                         | **3.0**                         | None                                                                                                                                                                                                                                                                     |
| Windows 7                                         | **2.0**                         | **3.0** (requires SP1, .NET Framework 4.0) <br /><br /> **4.0** (requires SP1, .NET Framework 4.5) <br /><br /> **5.0** (requires SP1, .NET Framework 4.5 and WMF 4.0) <br /><br /> **5.1** (requires SP1, .NET Framework 4.5.2 where WMF 3.0 **must not be installed**) |
| Windows Vista                                     | None                            | **1.0** <br /><br /> **2.0** (requires SP1)                                                                                                                                                                                                                              |
| Windows XP (NT 5.2 family) <br /> (x64 and IA-64) | None                            | **1.0** (requires .NET Framework 2.0) <br /><br /> **2.0** (requires SP2 and .NET Framework 2.0 SP1)                                                                                                                                                                     |
| Windows XP (NT 5.1 family) <br /> (x86 and IA-64) | None                            | **1.0** (requires SP2 and .NET Framework 2.0) <br /><br /> **2.0** (requires SP3 and .NET Framework 2.0 SP1)                                                                                                                                                             |
| Windows 2000 and older versions                   | None                            | None                                                                                                                                                                                                                                                                     |
